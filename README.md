# 🎨 מערכת מעקב צביעת קוביות

אפליקציית ווב למעקב אחרי ציור קוביות דקורטיביות.

## 🚀 שימוש מהיר

1. פתח את `index.html` בדפדפן (Safari, Chrome, Firefox וכו')
2. תמונת הקוביות תופיע באופן אוטומטי

## 📋 תכונות

✅ **גריד חזותי** - ראה את כל הקוביות בזמן אמת
✅ **רשימה מפורטת** - תצוגה מוקדשת עם צבעים ופרטים
✅ **מעקב התקדמות** - ראה כמה אחוזים הקובייות הושלמו
✅ **חיפוש** - חפש קובייה לפי מספר
✅ **הערות** - הוסף הערות לכל קובייה
✅ **שמירה טומנת** - הנתונים נשמרים בטלפון/מחשב שלך

## 🎨 הגדרת הנתונים שלך

עדכן את `data.js` עם הנתונים שלך:

```javascript
const CUBES_DATA = {
    1: { faces: [1, 2] },        // קובייה 1: טורקיז וורוד
    2: { faces: [3, 4] },        // קובייה 2: כחול כהה וכחול כהה מאוד
    3: { faces: [] },            // קובייה 3: טבעית
    // ... וכו'
};
```

### מספרי הצבעים:
- 1 = טורקיז בהיר
- 2 = ורוד
- 3 = כחול כהה
- 4 = כחול כהה מאוד / שחור
- 5 = נייבי
- 6 = כתום
- 7 = סגול
- 8 = צהוב
- 9 = כחול בהיר
- 10 = ירוק בהיר
- 11 = ירוק רך
- 12 = בורדו

## 📱 תצוגה מסתגלת

האפליקציה עובדת בטלפוני, טאבלטים ומחשבים.

## 💾 האם הנתונים נשמרים?

כן! הנתונים נשמרים בהתאם הדפדפן:
- סטטוס "הושלם" שלך
- הערות שלך

## 🔧 ללא דרישות של התקנה

אפליקציה זו פועלת בעיירת מלא - אתה לא צריך להתקין כלום!
פשוט פתח את `index.html` בדפדפן.

## 📞 תשובה לשאלות נפוצות

**ס: איך אני מוסיף/משנה קובייה?**
ת: ערוך את `data.js` והזן את הקבויות כמו בדוגמה למעלה.

**ס: האם אני יכול להשתמש בטלפון שלי?**
ת: כן! פשוט שלח לעצמך את הלינק או שמור את הקבצים על דיסק זיכרון.

**ס: איך אני מייצא את הנתונים?**
ת: הנתונים נשמרים ב-LocalStorage. אתה יכול להעתיק את הנתונים מ-DevTools Console או לבקש עדכון בעתיד.
